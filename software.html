Software

The code I produced during my PhD was written in Python and incorporated in GPy. This is a machine learning framework for Gaussian processes. It was initially developed by the Machine Learning research group of the University of Sheffield, but now it also has many contributors from different places.

The modules was mainly involved with were:
-- Vector-valued Gaussian processes (aka Coregionalization)
-- Expectation propagation
-- Mixed noise models

Here are a couple of tutorials I wrote for getting started with some of GPy models.
-- Vector-valued regression
-- Heteroscedastic regression
