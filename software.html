Software

The code I have produced during my PhD is written in python and incorporated in GPy. This is a machine learning framework for Gaussian processes and developed by the Machine Learning research group of the University of Sheffield.

The modules I am mainly involved with are:
-- Vector-valued Gaussian processes (aka Coregionalization)
-- Expectation propagation
-- Mixed noise models

Here are a couple of tutorials I have written for getting started with some of GPy models.
-- Vector-valued regression
-- Heteroscedastic regression
